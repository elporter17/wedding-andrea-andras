# Andrea & András — Wedding Invitation 💍

One-page wedding invitation site · **4 July 2026** · Szentendre, Hungary.

🇬🇧 / 🇭🇺 Bilingual (English / Hungarian) with a language switch.

## Features
- Animated preloader with the **A & A** monogram
- Soft scroll-reveal animations, floating decorations and **falling petals**
- "Where it all began" story section
- Live **countdown** to the big day
- Event details with **Google Maps** links to both venues
- **Add to calendar** (Google Calendar) button
- Photo **gallery with lightbox**
- Open Graph preview + favicon for sharing
- Fully responsive (desktop & mobile)

## Structure
```
index.html              → the website
invitation-pdf.html     → printable invitation (source for the PDF)
Andrea-Andras-Invitation.pdf → shareable 2-page PDF (EN + HU)
images/                 → photos
```

## Run locally
Any static server works, e.g.:
```bash
npx serve .
```
Then open the shown URL.
